# rtech | Digital Portfolio 🚀

Welcome to the repository for my personal digital portfolio, **rtech**. This project serves as a centralized hub to showcase my skills, educational background, and software development projects.

## 🌟 Overview
Built as a Single Page Application (SPA), this portfolio utilizes custom CSS and vanilla JavaScript to provide a fast, seamless browsing experience without page reloads. It features a modern, corporate-light aesthetic with organic gradient backgrounds.

## 🛠️ Built With
* **HTML5:** Semantic structuring and SPA layout.
* **CSS3:** Custom variables, Flexbox/Grid layouts, glassmorphism effects, and CSS animations.
* **Vanilla JavaScript:** DOM manipulation for tab-switching logic.

## 📂 Project Sections
1. **Home:** Introduction and quick-access navigation.
2. **Digital Portfolio:** Details about my background as an IT Student at Torres Capitol College, and my core technical skills.
3. **Websites:** Live links to web platforms I have built, including my **Online Voting System (E-Vote)**.
4. **Apps:** A dedicated space for my downloadable desktop applications (Currently under maintenance as new Python/Java builds are packaged).
